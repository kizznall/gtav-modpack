GTAV Modpack Installation Guide

Only for GTAV Legacy. Works on Steam and Rockstar Launcher.

1. Extract the Modpack

Unzip the downloaded modpack to any folder.

2. Locate Your GTAV Folder

Steam default:

C:\Program Files (x86)\Steam\steamapps\common\Grand Theft Auto V

Rockstar default:

C:\Program Files\Rockstar Games\Grand Theft Auto V

3. Copy Mod Files

Open the extracted WITH GTA.EXE folder and copy its contents into your GTAV folder.

4. Install ASI Manager

  1. Download OpenIV and install.
  2. Launch OpenIV → select Grand Theft Auto V (Windows).
  3. Tools → ASI Manager → Install all.
  4. Enable Edit mode (top-right).

5. Install .OIV Packages

In the modpack folder, open OIV and drag each .oiv file into the OpenIV window. Confirm installation via the Package Installer.

6. Install Custom gameconfig.xml

  1. Download gameconfig.xml from Assets.
  2. In OpenIV (Edit mode), navigate to:
    mods\update\update.rpf\common\data
  3. Drag gameconfig.xml here and overwrite (should show only Compressed; and ~89 KB).

7. Adjust Heap Size

Open HeapAdjuster.ini in GTA V folder. Change:

heapSizeMB=1024

to:

heapSizeMB=2048

8. Clean Up Mods (Optional)

Place gtavcleaner.exe in your GTAV folder. Run it to remove all mods.

For a visual guide, search YouTube: “Install GTAV mods with OpenIV”.
